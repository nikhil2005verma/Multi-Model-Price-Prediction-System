# 🚀 Multi-Model Price Prediction System

A Machine Learning project that combines **House Price Prediction (Bangalore)**, **Laptop Price Prediction**, and **Car Price Prediction** into a single unified application.

---

## 📌 Project Overview

This project is designed to predict prices across multiple domains using machine learning models. It provides a user-friendly interface where users can input details and get accurate price predictions.

### 🔹 Modules Included:

* 🏠 Bangalore House Price Prediction
* 💻 Laptop Price Prediction
* 🚗 Car Price Prediction

---

## 🎯 Features

* Multiple ML models integrated into one system
* Clean and interactive UI (Streamlit GUI)
* Real-time prediction based on user inputs
* End-to-end ML pipeline implementation
* Data preprocessing + model training + deployment

---

## 🧠 Technologies Used

* Python 🐍
* Pandas & NumPy
* Scikit-learn
* Streamlit
* Matplotlib / Seaborn (for visualization)

---

## 📊 Machine Learning Workflow

1. Data Collection
2. Data Cleaning & Preprocessing
3. Feature Engineering
4. Model Training
5. Model Evaluation
6. Deployment using Streamlit

## 🏠 House Price Prediction

### Input Features:

* Location
* Total Square Feet
* BHK
* Number of Bathrooms

### Output:

* Predicted House Price

## 💻 Laptop Price Prediction

* Predicted Laptop Price

## 🚗 Car Price Prediction

### Output:

* Predicted Car Price
